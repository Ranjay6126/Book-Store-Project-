# Book Store System

A full stack Book Store application built using the MERN stack. The project provides a RESTful API for managing books and a modern frontend for smooth user interaction.

---

## Project Duration

**June 2024 – July 2024**

---

## Features

* Create, update, view, and delete books using REST APIs
* Modular backend architecture with clean route and model separation
* Responsive and interactive frontend with smooth navigation
* Reusable UI components for better scalability

---

## Tech Stack

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* REST API
* CORS

### Frontend

* React.js
* React Router
* Tailwind CSS
* Axios
* JavaScript

---

## Backend Highlights

* Implemented RESTful CRUD APIs to manage book records
* Structured server logic using Express Router for maintainable code
* Applied CORS configuration to enable secure cross origin communication
* Designed modular models and routes for better scalability

---

## Frontend Highlights

* Built a single page application using React Router
* Developed reusable Book Card components and modal dialogs
* Integrated Axios for seamless API communication
* Enhanced user experience with interactive alerts and smooth UI flow

---

## API Endpoints (Sample)

```
GET    /api/books
POST   /api/books
PUT    /api/books/:id
DELETE /api/books/:id
```

